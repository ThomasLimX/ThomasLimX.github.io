Accessibility

I have taken into account accessibility by:

  1. I have chosen color contrast for visual impairments, my site use an appropriate color to contrast between my text and background elements. For example, I have use a dark text on a light background and vice versa in dark mode ensures that users with visual impairments or color blindness can comfortably read the content without straining their eyes. After viewing site 3, this is what I structure my colour scheme to ahere to.

  2. I have ensure that my website supports keyboard navigation through interactive elements such as links as button. This is critical for users who have limited motor abilities or rely on keyboards instead of a mouse. The navigation bar and scrollable elements that are designed with accessibility when I was making it. This is to allow users to access all content using the keyboard.

  3. Screen reader compatibility, I have included alternative text for all my images and labels for form elements. This will allow my website is accessible to users who rely on screen readers. This will help the visually impaired users that understand the content and navigate the site effectively.

Usability

I considered the usability of my site in the following ways:

  1. I have allowed my site to adapt to various screen sizes, to ensure a smooth experience on both desktop and mobile devices. This responsiveness is critical for usability as it allows users on different devices to access the site without any functionality or content loss. Allow users to view the content fully without lacking any information on different devices.

  2. My navigation menu is simple and intuitive with clear labeled links to different sections like "Homes" , "Projects", and "Contact". This ensures that users can quickly find the information that they need without looking through every single page of the website.The mobile menu uses a toggle for easy access, improving usability for mobile users.

  3. I have also implemented a theme toggle feature, allowing users to switch between dark and light modes. This allow me to enhances visual comfort for users in different lighting environmenets but also makes the site more accessible and usable for my users with specific visual preferences. Which mode is more pleasing to their eyes or which mode does not strain their eyes.
